# **Easy-Carousel-Stack**

### This is very easy way usable light weight carousel!

## **Supported Properties**
 - **Duration (number)**
 - **Loop (boolean)**
 - **Indicate (boolean)**
 - **Alternate (boolean)**
 - **Custom CSS Class (string)**

 ## **Usage Example**
